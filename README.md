# School Helpdesk System (🚧 In Development)

## 📌 Overview

The **School Helpdesk System** is a web-based application designed to manage and resolve student and staff queries efficiently.
This project is currently under development and aims to streamline communication between users and administration.

---

## 🚀 Features (Planned / In Progress)

* 📝 Ticket/Issue submission system
* 👨‍💼 Admin dashboard for managing requests
* 👨‍🎓 Student/User login system
* 📊 Status tracking of issues
* 🔔 Notifications (future scope)

---

## 🛠️ Technologies Used

* Language: Java
* Framework: (e.g., Spring Boot / Servlet / JSP)
* Database: (MySQL / PostgreSQL / etc.)
* Frontend: HTML, CSS, JavaScript

1. Clone the repository:

```bash
git clone https://github.com/dshrijal/School-helpdesk-Systemn.git
```

2. Navigate to the project directory:

```bash
cd School-helpdesk-Systemn
```

3. Run the project using your IDE or server.
